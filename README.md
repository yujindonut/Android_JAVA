# Android_MemoApp

### 개요

빵 맛집을 기록하기 위한 간단한 앱!

### 개발

- 소요 기간 : 5일
- 사용 툴 : Android Studio

### 주요기능

- 맛집 리스트 - `ListAdapter`
- CRUD기능 ( 추가, 수정, 삭제 , 검색 ) - `SQliteDB` , `SQL`

### 아쉬웠던 점
- 👹 사진 찍어서 저장할 수 있도록 하는 기능을 추가
- 👹 더 예쁜 UI
- 👹 검색하는 기능에서, 나의 저장목록을 검색하는 것보다, api를 사용하여, 실제 근처 맛집을 검색하고 이를 갖고오는 화면이 있었으면!
